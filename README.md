# Workout
Web Applications Final project

## Features

- Flask backend with SQLite for tracking workouts
- Dynamic workout generation by category (legs, arms, chest, etc.)
- User progress tracking with PRs and goals
- Stylish dark mode with ultraviolet accents

## Demo Instructions

1. `python init_db.py` to seed the database
2. `python app.py` to start the server
3. Visit `http://127.0.0.1:5000/`
